CredResolve – MERN Expense Sharing Application

CredResolve is a MERN-stack expense sharing application that allows users to create groups and add shared expenses.
The project focuses on backend correctness, API design, and frontend integration with a clean and minimal UI.

Prerequisites

Ensure the following are installed on your system:

Node.js (v18 or later recommended)

MongoDB (MongoDB Atlas or local MongoDB)

npm

Environment Setup

Create a .env file in the project root (CredResolve/):

PORT=5000
MONGO_URI=your_mongodb_connection_string


⚠️ Do not commit the .env file to version control.

Backend Setup

Navigate to the project root:

cd CredResolve


Install backend dependencies:

npm install


Start the backend server:

npm run dev


Backend runs at:

http://localhost:5000

Frontend Setup

Navigate to the frontend directory:

cd frontend


Install frontend dependencies:

npm install


Start the development server:

npm run dev


Frontend runs at:

http://localhost:5173


CredResolve demonstrates:

MERN stack fundamentals

RESTful API design

MongoDB data modeling

Frontend–backend communication

The project is structured to be simple, extensible, and production-aligned.
